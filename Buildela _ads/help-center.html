Ad Flight Schedule

Introducing Ad Flight Schedule Beta

Establish specific start and end dates for your creatives within Buildela Ads. This powerful scheduling tool empowers you to seamlessly modify your creative messages during the campaign, ensuring precise control within predetermined timeframes.

Setting up a flight schedule involves setting one or more start and end dates by the hour (down to the quarter hour) for each ad assigned to a line.

How to Set up Ad Flight Schedule:

For each creative, you can set a START DATE & TIME and an END DATE & TIME. Ads will have a “pending start date” status before the Start Date & Time arrives. Once the End Date & Time arrives, the ad will switch to paused.

  1. Create your creative.
  2. Save your creative.
  3. Click Edit, for the creative that needs an Ad Flight Schedule.
  4. Confirm you are in the Edit Creative pane. Select “specify a Start and end date”
  5. Choose the Specify a start and end date radial button.
  6. Set the Start Date and the End Date.
  7. Pick the Time Zone.
  8. Hit Save

Note: We'll schedule your ad to run based on the selected time zone. By default your account time zone is selected

When an Ad Flight Schedule is in place, two new columns for start dates and end dates are added to the Creatives tab. They will show the number of them.
